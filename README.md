# Crossword Bot 🧩

Telegram-бот для игры в кроссворд с друзьями. Генерирует кроссворды через Google Gemini, рисует сетку картинкой, ведёт рейтинг.

## Что умеет (этап 1 — скелет)

Сейчас бот умеет только запускаться, отвечать на `/start`, `/help`, `/правила` и `/chatid`. Игровая логика будет добавлена на следующих этапах.

## Установка локально (для проверки)

### 1. Клонировать проект и поставить зависимости

```bash
cd crossword_bot
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

pip install -r requirements.txt
```

### 2. Настроить `.env`

Скопируй `.env.example` в `.env` и заполни:

```env
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_ID=твой_telegram_id
GEMINI_API_KEY=твой_ключ_gemini
CHAT_ID=  # пока пусто, заполнишь после шага 4
```

### 3. Запустить бота

```bash
python -m bot.main
```

Должно появиться:
```
[INFO] Бот запускается...
[INFO] Запущен как @твой_бот (id=...)
```

### 4. Получить ID чата

- Добавь бота в группу с друзьями (дай права админа)
- Напиши в группе `/chatid`
- Бот пришлёт ID — скопируй его в `.env` как `CHAT_ID=...`
- Перезапусти бота
